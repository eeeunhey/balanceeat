# 🥗 Balance Eat (밸런스일조)

> 🍽️ **식단 관리를 더 편하게, 다음 끼니의 고민까지 덜어주는 서비스**

**Balance Eat**은 사용자가 입력한 식단을 기반으로  
영양 정보를 자동 분석하고, 이해를 돕는 코멘트와 함께  
**다음 끼니까지 자연스럽게 추천해주는 식단 관리 서비스**입니다.

## 🟢 Problems We Solved

밸런스일조가 해결하고자 했던 핵심 문제는 다음과 같습니다.

- **POINT 01**  
  직접 계산하지 않아도 **영양 상태를 한눈에 파악**할 수 있도록

- **POINT 02**  
  단순 기록이 아닌, **이해를 돕는 분석 코멘트** 제공

- **POINT 03**  
  현재 영양 상태를 기준으로 **다음 끼니를 자연스럽게 추천**

- **POINT 04**  
  사용자가 신경 쓰지 않아도 **식단 관리가 흐름처럼 이어지도록 설계**

---

## ✨ Key Features

- 식단 입력 및 기록
- 끼니별 / 하루 단위 영양 분석
- AI 기반 영양 분석 코멘트 제공
- 부족하거나 과한 영양소 판단
- 영양 상태 기반 다음 끼니 추천
- 로컬 스토리지 기반 기록 유지 (서버 없이 사용 가능)

---

## 🧱 Tech Stack & API Integration

### 🥗 Nutrition API
- **농촌진흥청 국립식량과학원 – 농식품 식단관리 API**
- 음식별 영양 성분 조회 (칼로리, 탄단지, 나트륨 등)
- 사용자 입력 식단의 **끼니별 / 하루 단위 분석을 위한 기초 데이터**로 활용

### 🤖 AI
- **Gemini AI API**
- 하루 식단 데이터를 기반으로:
  - 영양 분석 코멘트 생성
  - 부족/과다 영양소 판단
  - 다음 끼니 추천
- 프로젝트의 핵심 흐름:  
  **기록 → 분석 → 인사이트 제공**

### 💾 Data Storage
- **Local Storage 기반 저장**
- 서버 없이도 사용자 식단 기록 유지
- 추후 DB 연동을 고려한 구조로 설계

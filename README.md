# 🥗 Balance Eat

영양 균형을 기반으로 **식단을 분석하고 추천하는 서비스**입니다.  
사용자가 입력한 식단 기록을 바탕으로 **영양 요약·피드백**을 제공하고,  
부족한 영양소를 보완할 수 있는 **추천 식단**을 제안합니다.

---

## ✨ Key Features

- 식단 입력 및 기록
- 하루 식단 영양 요약 & 피드백 제공
- 부족한 영양소 기반 추천 메뉴
- 목표 설정을 통한 식단 관리

## 🧱 Tech Stack

**Frontend**
- React
- JavaScript / TypeScript
- CSS

**Backend / API**
- Open API (Nutrition / Food Data)

**AI**
- Gemini API 기반 식단 요약 및 피드백

---

## 🗺️ Service Flow

1. 사용자가 식단을 입력
2. 영양 성분 데이터 분석
3. AI 기반 식단 요약 및 피드백 생성
4. 부족 영양소 기반 추천 제공
